Die Datei "Antiterra.apk" ist die kompilierte Version des Spiels.

Das Spiel als Video kann mit der "Videovorstellung Antiterra.mp4" angesehen werden.

---------------------------------------------------------------------------------------------------
Systemanforderungen an das Smartphone: Android Version 7 oder höher
				       API Level 24 oder höher
				       Internetverbindung (Sim-Karte verwenden, wenn möglich)
---------------------------------------------------------------------------------------------------

Installation des Spiels mit Windows. 

1. In den Einstellungen des Smartphones die Installation von unbekannten Apps erlauben.

2. Das entsperrte Handy per USB-Kabel mit dem Rechner verbinden. 

3. Über den Datei-Explorer von Windows auf den Speicher des Smartphones zugreifen.

4. Einen neuen Ordner "Games" im Speicher des Smartphones erstellen.

5. Die Datei "Antiterra.apk" in den Games-Ordner kopieren.

6. Auf dem Smartphone über "Dateien" auf den internen Speicher und den Ordner "Games" öffnen.

7. Die Datei "Antiterra.apk" öffnen und installieren. (Das Smartphone kann vom Rechner getrennt werden)

8. Beim Spielen die Verwendung der Kamera und der Verwendung von Ortungsdiensten zustimmen.

---------------------------------------------------------------------------------------------------

Installation mit MAC OS

1. In den Einstellungen des Smartphones die Installation von unbekannten Apps erlauben.

2. Die AndroidFileTransfer.dmg auf dem Mac installieren. (Quelle: https://www.android.com/filetransfer/)

3. Das entsperrte Handy per USB-Kabel mit dem Rechner verbinden.

4. Android File Transfer auf dem Mac öffnen.

5. Einen neuen Ordner "Games" im Speicher des Smartphones erstellen.

6. Die Datei "Antiterra.apk" in den Games-Ordner kopieren.

7. Auf dem Smartphone über "Dateien" auf den internen Speicher und den Ordner "Games" öffnen.

8. Die Datei "Antiterra.apk" öffnen und installieren. (Das Smartphone kann vom Mac getrennt werden)

9. Bei der Verwendung die Verwendung der Kamera und der Verwendung von Ortungsdiensten zustimmen.

---------------------------------------------------------------------------------------------------

Installation über Googledrive

1. In den Einstellungen des Smartphones die Installation von unbekannten Apps erlauben.

2. "Antiterra.apk" auf einem Googledrive hochladen.

3. Dasselbe Googledrive auf dem Smartphone öffnen.

4. Die Datei "Antiterra.apk" herunterladen und installieren.

5. Bei der Verwendung die Verwendung der Kamera und der Verwendung von Ortungsdiensten zustimmen.
